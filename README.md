# KNN Regression Project

## Description

This project demonstrates the implementation of K-Nearest Neighbors (KNN) regression on various datasets. It includes data preprocessing, model training, and comparison with other regression algorithms such as Linear Regression, Decision Tree, and Random Forest.

## Files

- `task1.ipynb`: Jupyter notebook containing the analysis for Task 1, including data loading, preprocessing, model training, and evaluation.
- `knn.ipynb`: Jupyter notebook focused on KNN regression implementation and analysis.
- `task1_dataset.csv`: Dataset used in Task 1.
- `knn_regression_dataset.csv`: Dataset for KNN regression.
- `KNN.txt`: Notes and key points about KNN algorithm.
- `requirements.txt`: List of Python dependencies.
- `README.md`: This file.

## Installation

1. Clone or download the repository.
2. Install the required packages:

   ```bash
   pip install -r requirements.txt
   ```

## Usage

1. Open the Jupyter notebooks (`task1.ipynb` or `knn.ipynb`) in Jupyter Lab or Notebook.
2. Run the cells in order to execute the code.
3. The notebooks include data visualization, model training, and performance metrics.

## Dependencies

- pandas
- numpy
- scikit-learn
- matplotlib
